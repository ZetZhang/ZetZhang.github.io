vim spf13 + 个人配置(vim-plug)

Posted on 2019-12-07 | In 2019.12

vim方案

  • nerdtree：侧边显示目录
  • nerdcommenter：注释插件
  • tagbar：显示文档结构
  • fugitive：git功能
  • ctrlp：ctrlp搜索功能
  • easymotion：光标跳转
  • YouCompleteMe：自动补全工具
  • fzf：命令行搜索工具
  • rking/ag.vim：也是个搜索工具
  • auto-pairs：自动补全引号、圆括号、花括号等
  • vim-airline：状态栏美化
  • vim-airline-themes
  • universal-ctags：ctags
  • asyncrun.vim：异步执行后台
  • rainbow：彩虹括号
  • indentline：缩进线
  • w0ng/vim-hybrid：颜色主题
  • vim-colors-solarized：颜色主题
  • vim-gutentags：自动增量更新ctags索引
  • NeoSolarized：支持真彩色的solarized颜色方案
